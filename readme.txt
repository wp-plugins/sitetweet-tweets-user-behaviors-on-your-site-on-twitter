This plugin allows blogs to post user behaviors on their site straight to their Twitter account.

== Description ==

[sitetweet](http://sitetweet.net "sitetweet Twitter site behavior") allows webmasters to post user behaviors to their Twitter account.  Behaviors such as:

* user visited a certain page
* user did a specific search
* user converted

on your blog can be posted straight to your Twitter account with sitetweet.  

Related Links:

* [sitetweet Plugin homepage](http://sitetweet.net/wordpress "sitetweet plugin homepage")
* [sitetweet homepage](http://sitetweet.net "sitetweet")


== Installation ==

Installation is simple 5 step process:

1. Login or [create a new sitetweet account](http://sitetweet.net/newuser "create a new sitetweet account") and retrieve your API key from your [sitetweet profile](http://sitetweet.net/profile "sitetweet profile")
1. Download and unzip the plugin
1. Upload the sitetweet plugin directory, along with its contents, to your WordPress plugins directory: <code>/wp-content/plugins/</code>. 
1. Activate the plugin.
1. Select the <code>sitetweet</code> option from the <code>Settings</code> menu and input your API key from step 1
</ol>

Once you have installed the plugin, simple add a <strong>Custom Field</strong> to your posts and/or pages named <code>sitetweet</code>. You can then
specify what you want your Twitter message to display in the <code>Value</code> box.
